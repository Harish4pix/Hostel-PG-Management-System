Hostel-PG-Management-System

This is a Hostel/PG Management System developed using Java Swing. The project provides an interactive and user-friendly GUI to efficiently manage hostel/PG operations such as student/tenant details, room allocation, payments, and room availability. The interface is clean, structured, and designed to simplify management tasks for administrators.

Features

Add, update, and delete student/tenant details

Room allocation and deallocation

Track room availability & occupancy

Maintain payment records

View payment history of each tenant

Search and filter records

Attractive and responsive Swing-based GUI

Separate modules/screens for easy navigation

Validation for input fields

Supports multiple rooms and categories (e.g., AC/Non-AC)

Technologies Used

Java (Core Java)

Java Swing – GUI

Java AWT – Event Handling

Java JDBC – Database Connectivity

MySQL / SQLite – Backend Database

Object-Oriented Programming (OOP) concepts

How It Works

Administrator launches the application

Home dashboard appears with navigation options

Admin can:

Add new tenant

Allocate a room

Check available rooms

Record payments

Update or delete a record

All data is saved and retrieved from the database

Payment and room details can be viewed anytime through GUI panels

How to Run

Set up the database (MySQL/SQLite) and import the required tables

Compile all Java files:

javac *.java


Run the main file (e.g., MainDashboard.java or your starting file):

java MainDashboard


The GUI window will open and you can begin using the system
